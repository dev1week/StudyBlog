상위문서

MarkDown

개요

  • 옵시디언에서 시각적 요소를 커스텀하기 위한 Snippets입니다.
  • 텍스트 색상, 폰트 크기, 간격, 배경색 등 다양한 스타일링 옵션을 사용자가 직접 조정할 수 있습니다.

적용하기

일반적인 방법

1. CSS Snippets 폴더로 이동

  • 옵시디언 설정에서 Appearance 섹션으로 이동하여 CSS Snippets 부분을 찾습니다.
  • 폴더 아이콘을 클릭하면 스니펫을 저장할 폴더가 열립니다.

2. CSS 파일 생성

  • 윈도우 사용자
    • 스니펫 폴더 안에서 우클릭하여 새파일을 선택합니다.
    • 파일의 확장자를 .txt 에서 .css 로 변경합니다.
    • 이 파일 안에 원하는 css 를 입력합니다.
  • 맥 사용자
    • 스포트라이트 검색을 사용하여 텍스트 편집기를 열고 새 문서를 생성합니다.
    • cmd + shift + t 단축키를 사용해 텍스트 파일로 설정한 다음, 파일을 .css 확장자로 저장합니다.
    • 저장한 파일을 CSS 스니펫 폴더로 이동시키면 됩니다.

3. CSS 스니펫 적용하기

  • CSS 파일을 CSS Snippets 폴더에 저장합니다.
  • 옵시디언 설정에서 CSS 스니펫 섹션으로 이동합니다.
  • 새로 생성한 CSS 파일을 활성화합니다.

CSS Editor 플러그인

1. CSS 파일 생성

  • 플러그인 설치후 cmd/ctrl + p 를 눌러 명령어 커맨드를 실행합니다.
  • Create CSS Snippets 를 클릭합니다.
  • 파일 이름을 작성합니다.
  • Vault 이름/./obsidian/snippets 에 CSS 파일이 생성됩니다.

2. CSS 파일 수정

  • edit CSS Snippets를 실행하여 수정할 CSS 파일을 열어 코드를 수정할 수 있습니다.
